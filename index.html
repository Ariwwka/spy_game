How to play

  1. Choose which character packs to include, then tap Continue.
  2. Pick the number of players and tap Start Game.
  3. The game secretly picks one character. Most players get that character — some randomly become a SPY.
  4. Pass the phone around. Each player privately reveals their card, then hides it.
  5. Spies only see the word SPY — they don't know the character!
  6. Discuss and ask questions to find the spies. Spies try to blend in.

There's no fixed number of spies — a round might have none, one, or many. Each player has a 30% chance of being a spy. You can tap ✕ Menu anytime to start over.
